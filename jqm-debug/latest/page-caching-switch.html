dom-cache=" "

  1. Press the 'true' or 'false' button below.
  2. Check the set dom-cache value through the dom-cache="what" message in header.
  3. Press 'Add new DOM' button. (To check whether keeping previously-visited page in the DOM, 'New DOM is added.' message is added in this page.)
  4. Press the 'Go external page' button and press the 'Back' button of the top of the 'External' page.
  5. Check whether the DOM is kept when revisiting this page. :
    • If the page is set to 'dom-cache="true"':
      Check whether the 'New DOM is added.' message is kept.
    • Or if the page is set to 'dom-cache="false"':
      Check whether the 'New DOM is added.' message is NOT kept and is removed.
